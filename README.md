#### 介绍
预备做一个网页。
它是一个类似于excel的东西。格子里可以写3。3是作为js代码的3。当焦点离开格子的时候进行求值。可以在一个格子里寻址另一个格子的求值结果。一个格子代码的（不是非要有的）副作用可以寻址并赋值另一个格子，这样fetch下来的数据就可以用来填格子了。但是这样每个格子好像还有“表里”的二分似的，就和excel的格子具有数值与公式的二相性一样。
它是一个极为通用、灵活、轻量的东西。
